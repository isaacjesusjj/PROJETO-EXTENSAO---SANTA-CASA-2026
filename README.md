# Rede Liga do Bem

Site institucional desenvolvido para a ONG **Rede Liga do Bem**, com o objetivo de apresentar a instituição, suas atividades, formas de doação, localização e canais de contato.

## Sobre o projeto

A Rede Liga do Bem é uma organização comunitária sem fins lucrativos que promove atividades sociais, esportivas e educativas para crianças, jovens e famílias.  
O site foi criado como parte de um projeto acadêmico, buscando divulgar o trabalho da instituição e facilitar o contato com apoiadores, voluntários e pessoas interessadas em contribuir.

## Páginas do site

O projeto possui as seguintes páginas:

- **Início**: apresentação geral da Rede Liga do Bem.
- **Instituição**: história, missão, visão, valores e membros.
- **Atividades**: modalidades oferecidas, como futebol, vôlei e capoeira.
- **Doações**: formas de contribuir com Pix, itens físicos, voluntariado e parcerias.
- **Fale Conosco**: formulário funcional com envio de mensagem pelo WhatsApp, além de e-mail, telefone e localização.

## Melhorias aplicadas

O projeto passou por uma revisão geral para melhorar estrutura, organização, responsividade e funcionamento.

Principais ajustes realizados:

- Correção de links internos quebrados.
- Padronização do header e footer.
- Criação de CSS global.
- Criação de JavaScript global.
- Remoção de `onclick` inline.
- Menu mobile funcional e acessível.
- Formulário de contato funcionando via WhatsApp.
- Melhorias de responsividade para celular, tablet e desktop.
- Correção de textos e padronização visual.
- Adição de meta tags básicas de SEO.
- Melhorias de acessibilidade, como `alt` em imagens e atributos `aria`.
- Organização visual dos cards, botões, seções e footer.
- Correção do link do card “Fale Conosco”.
- Ajustes em imagens para evitar distorções.
- Padronização de cores, espaçamentos e botões.

## Como testar o projeto

Para testar localmente:

1. Clone o repositório:

```bash
git clone https://github.com/isaacjesusjj/PROJETO-EXTENSAO---SANTA-CASA-2026.git
```

2. Abra a pasta do projeto no VS Code.

3. Abra o arquivo `index.html` no navegador ou use a extensão **Live Server**.

4. Teste as páginas:

* Início
* Instituição
* Atividades
* Doações
* Fale Conosco

5. Verifique também:

* Menu mobile.
* Links do header.
* Links do footer.
* Cards da página inicial.
* Formulário de contato.
* Link do WhatsApp.
* Link do Instagram.
* Link do Google Maps.
* Responsividade em telas pequenas e grandes.
